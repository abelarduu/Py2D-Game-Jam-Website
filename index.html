Py2D Game Jam

Game jam de python para games 2D.

Transforme suas ideias em jogos 2D incríveis!

Participe da Py2D Game Jam e mostre suas habilidades em Python criando jogos com gráficos pixelados e mecânicas únicas.

Sobre

Esta Game Jam é voltada para o desenvolvimento de jogos 2D em pixel art, utilizando bibliotecas de Python. O foco é promover criatividade e aprendizado. Você pode usar bibliotecas como Pygame, Pyxel, Arcade e outras. O desafio é criar algo único e técnico, colaborando com outros desenvolvedores!

Requisitos

Para participar da Py2D Game Jam, você precisará atender aos seguintes requisitos importantes do evento:

Experiência básica em desenvolvimento com Python
Conhecimento de bibliotecas para desenvolvimento de jogos
Capacidade de criar e trabalhar com gráficos em pixel art
Trabalho colaborativo em equipe ou individual
Cumprir o prazo do evento, que será de 7 dias
Compromisso com as diretrizes de conduta e fair play
